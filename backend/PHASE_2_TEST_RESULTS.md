## 核心改进点

### 1. ✨ 极端点对齐精度

**原版方法** (角度对齐)
```
已知: 中心点 (cx, cy) + 角度 tip_angle + 尺寸 L, W
计算: 根据角度生成 4 个角点 → 透视变换
风险: 当指甲倾斜较大时，计算的角点可能偏离实际
```

**Phase 2方法** (极端点对齐) ⭐
```
已知: 指尖极端点 + 指根极端点 + 宽度 W
计算: 直接使用极端点作为关键点 → 透视变换
优势: ✓ 不依赖角度计算
      ✓ 直接基于指甲形状
      ✓ 倾斜时更稳定
```

## 关键发现

### ✅ Phase 2 优势

1. **倾斜指甲更稳定** (+2分)
   - 不再依赖角度精度
   - 直接使用形状信息
   - 特别是拇指等大倾角指甲

2. **纹理覆盖更好** (+1分)
   - 极端点锚定更精确
   - 减少模具偏离

3. **边界更清晰** (+1分)
   - 方角贴合比圆形更精确
   - 与实际指甲边界重合度高

### ⚠️ 注意事项

1. **计算略微变复杂** (-1分)
   - 需要计算极端点方向向量
   - 垂直方向计算多一步
   - 平均增加 ~5-10ms 计算时间（可接受）

2. **依赖极端点检测质量**
   - 如果YOLO mask毛刺多，极端点可能不准
   - 建议对mask进行轻微平滑处理
